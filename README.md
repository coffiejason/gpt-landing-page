
# GPT3 LANDING PAGE

A simple landing page for Open AI's GPT3 with email input validation

## Tech Stack

React, CSS


## Run Locally

Clone the project

```bash
  git clone https://github.com/coffiejason/gpt-landing-page
```

Go to the project directory

```bash
  cd project-directory
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
